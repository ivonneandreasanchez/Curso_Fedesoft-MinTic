JAVA SCRIPT

lenguae estático:
-declaración de tipo explícita
-fuertemente tipado: declaración de variables
-compila siempre, requiere ejecutables
-ejemplos:C, C++, java
lenguaje dinámico (scripting)
-declaración de tipo implicita
-debilmente tipado:no requiero especificar el tipo de dato
-verificación en tiempo de ejecución
-ejemplo: java script, ruby python, php
